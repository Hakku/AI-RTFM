# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it through [GitHub Security Advisories](https://github.com/Hakku/AI-RTFM/security/advisories/new). This allows us to assess and address the issue privately before public disclosure.

**Do not** open a public issue for security vulnerabilities.

## Scope

This policy covers the AI-RTFM repository contents: templates, kernels, tooling scripts, and configuration files.

## Response Timeline

- **Acknowledgment:** within 3 business days
- **Assessment:** within 7 business days
- **Fix or mitigation:** based on severity, targeting 30 days for critical issues
